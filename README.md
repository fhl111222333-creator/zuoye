# Omni-Union Core 2026 Starter Code

本项目包含：

- 多处 `TODO`：需要你补全功能；
- 多处隐藏 bug：需要你通过测试和调试发现；
- 部分过期说明：不要盲目信任注释和旧文档。

## 编译运行

```bash
cmake -S . -B build
cmake --build build
./build/campus_ops data
```

Windows + Visual Studio：用“打开本地文件夹 / Open a local folder”打开本文件夹，不要新建空项目复制代码。

## 输出文件

程序运行后会在当前工作目录生成：

- `result.txt`
- `error_log.txt`
- `repair_log.txt`
- `summary.txt`
